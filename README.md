# 🦀 100 Days of Rust

Welcome to my Rust learning journey! This repository serves as a public log of my progress, code experiments, and projects as I dive deep into the language over the next 100 days.

## 🎯 The Objective
My primary goal is to master Rust to build high-performance, memory-safe systems. This challenge is an intellectual commitment to level up my engineering capabilities, with a specific focus on conquering advanced concepts like:
*   Ownership and the Borrow Checker
*   Lifetimes
*   Fearless Concurrency

## 🚀 The Plan
To maintain momentum and ensure consistent growth, I am committing to:
1.  **Daily Code:** Writing and pushing Rust code every single day to maintain an unbroken activity streak.
2.  **Progressive Complexity:** Moving from basic syntax and `rustlings` exercises to building CLI tools and distributed systems.
